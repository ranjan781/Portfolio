# 🚀 Ranjan Yadav — Developer Portfolio

[![Live Demo](https://img.shields.io/badge/Live%20Demo-ranjan781.github.io-6C63FF?style=for-the-badge&logo=github)](https://ranjan781.github.io/portfolio)
[![Made With](https://img.shields.io/badge/Made%20With-HTML%20CSS%20JS-00E5FF?style=for-the-badge)](https://github.com/ranjan781/portfolio)

> Premium animated developer portfolio — built with pure HTML, CSS & Vanilla JavaScript. No frameworks. No libraries.

## ✨ Features

- 🌑 Dark glassmorphism design
- 🎆 Live particle network background  
- ✍️ Typing effect with multiple roles
- 📊 Real GitHub stats integration
- ⬇️ One-click resume download (PDF embedded)
- 🎠 Auto-sliding testimonial carousel
- 📱 Fully responsive (Desktop · Tablet · Mobile)
- ♿ Accessible (ARIA labels, keyboard nav)
- ⚡ Optimised animations via IntersectionObserver + requestAnimationFrame

## 🛠️ Tech Stack

`HTML5` `CSS3` `Vanilla JavaScript` `GitHub Pages`

## 📂 Structure

```
portfolio/
│
├── index.html        ← Entire portfolio (HTML + CSS + JS in one file)
├── .nojekyll         ← Disables Jekyll processing on GitHub Pages
└── README.md         ← This file
```

## 🚀 Deployment

Deployed via **GitHub Pages** — auto-deploys on every push to `main`.
